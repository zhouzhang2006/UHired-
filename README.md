# UHired?
An AI-powered resume analysis platform that evaluates resumes against job descriptions and provides personalized feedback to improve ATS compatibility and interview readiness

# Project Phases
Phase 1: Create frontend + backend project structure

Phase 2: User authentication (signup/login)

Phase 3: Resume upload (PDF/DOCX)

Phase 4: Resume parsing

Phase 5: Job description input

Phase 6: ATS scoring + keyword matching

Phase 7: AI feedback generation

Phase 8: Dashboard/results UI

Phase 9: Deployment

# Frontend, Backend, and Database
Frontend: React + Vite, Tailwind CSS, Axios. React Router

Backend: Node.js, Express, JWT, Multer

Database: MongoDB Atlas

AI API: OpenAI
